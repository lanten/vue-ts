# vue-ts

> 使用 `vue-cli` 创建

- Typescript
- Vue
- Vuex
- Vue-Router
- Less
- Eslint
- Jest

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run dev
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

### Run your unit tests
```
yarn run test:unit
```
